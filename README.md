# Lista de Usuários

Sistema FullStack de Gestão de Usuários.

# Como rodar o sistema

##### Rode os comandos abaixo no terminal para instalar as dependencias e rodar o servidor:

#### : Backend


```shell
cd backend
```

```shell
npm i
```

```shell
npm run dev
```

#### : Frontend


```shell
cd frontend
```

```shell
npm i
```

```shell
npm run dev
```

#### :Tecnologias

- Vuejs
- TypeScript
- NodeJS
